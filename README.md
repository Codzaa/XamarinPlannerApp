## Getting Started

```bash
Install Visual Studio 2019 or Later

Install Xamarin Forms
```

```bash

cd XamarinPlannerApp

```
```bash
Install Android sdk
```
![](pics/android.gif)
```bash
Install Emulator or Plug in Android Phone
```
![](pics/emulator.gif)

Build Application

Start Application
